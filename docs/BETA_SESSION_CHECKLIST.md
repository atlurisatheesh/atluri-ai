# BETA SESSION CHECKLIST
## Print This and Use It During Each Session

---

## PRE-SESSION (5 min before)

```
□ Test your mic/camera
□ App open in browser
□ Notepad ready
□ Slack/email muted
□ Water nearby
```

---

## OPENING (Read This)

> "I need to find where this breaks. You can't hurt my feelings. If something sucks, tell me. That's the most valuable thing you can give me."

---

## 30-MINUTE STRUCTURE

| Time | Do This |
|------|---------|
| 0-5 | Setup, explain format, mic test |
| 5-10 | Easy questions: "Tell me about yourself" |
| 10-15 | Hard: "Tell me about a time you failed" |
| 15-20 | Stress: Interrupt them. Stay silent. Add pressure. |
| 20-25 | Ask: "How are you feeling? Is this helping?" |
| 25-30 | Debrief questions below |

---

## DEBRIEF QUESTIONS (Ask These Exactly)

1. **Would you use this in a REAL interview tomorrow?** (Not could — would)
2. **What was the most frustrating moment?**
3. **Did anything make you trust it LESS?**
4. **One thing that would make you trust it more?**

---

## OBSERVE & RECORD

```
□ Did they look at suggestions?      Y / N
□ Did they follow suggestions?       Y / N
□ Did they seem less stressed?       Y / N
□ Would they use it for real?        Y / N
□ What broke?                        _______________
□ What confused them?                _______________
□ Exact quote about trust:           "_______________"
```

---

## RED FLAGS (Stop and Fix)

- They say "I don't trust this"
- They seem MORE stressed
- System freezes >3 seconds
- Same bug twice
- They give up

---

## AFTER 10 SESSIONS — GO / NO-GO

| Check | Target | Actual |
|-------|--------|--------|
| "Would use in real interview" | ≥50% Yes | ___ |
| Session completion | ≥70% | ___ |
| Return for session 2 | ≥40% | ___ |
| Trust score average | ≥60 | ___ |
| Critical bugs open | ≤2 | ___ |

**Pass 5/5 = GO**  
**Pass 3-4 = FIX AND RETEST**  
**Pass <3 = NO-GO**

---

## THE ONLY QUESTION THAT MATTERS

> "Would you use this in a real interview tomorrow?"

If <50% say yes after 10 sessions: **Don't launch.**
