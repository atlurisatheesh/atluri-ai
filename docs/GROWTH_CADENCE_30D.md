# Growth Cadence — 30 Day Operating Board

## Objective
Run a repeatable weekly distribution loop tied directly to offer-probability outcomes.

## Weekly Publishing Checklist

### Monday — Insight + Pattern
- Publish one post on the top failure pattern from recent sessions.
- Include one anonymized “before vs after” perception delta.
- CTA: run one pressure round now.

### Tuesday — Product Proof
- Publish a short clip of live perception signals changing in-session.
- Include one concrete metric-backed outcome.
- CTA: view the improvement snapshot report.

### Wednesday — Founder Authority
- Publish one “what changed” narrative from a meaningful offer-probability gain.
- Include top negative driver and the fix that moved it.
- CTA: book a trial round.

### Thursday — Educational Breakdown
- Publish one tactical framework post (STAR, ownership, impact density, risk drift control).
- Include an actionable 3-step mini drill.
- CTA: run the drill in-app.

### Friday — Social Proof + Conversion
- Publish one testimonial or user quote tied to a measurable delta.
- Include one screenshot of public-safe shared snapshot.
- CTA: start your first pressure round.

## Weekly KPI Review Template
- What shipped this week (tickets/features)
- Top conversion metric movement (landing CVR, trial->first round)
- Offer-probability KPI movement:
  - median delta this week
  - % users with >=3 sessions and positive delta
- Top positive driver observed
- Top negative driver observed
- Highest-leverage change for next week

## Content Board Fields
- Channel
- Post type (pattern/proof/authority/education/social)
- Core claim
- Supporting evidence
- CTA
- Owner
- Publish date
- Status (draft/scheduled/published)
- Outcome metric after 48h
